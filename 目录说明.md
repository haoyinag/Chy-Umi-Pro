## AngSi 新配送系统

> 基于 REACT+UMI3.X+TS+ANTD4.X 构建

#### 目录说明

> 各个目录下有对应的`XXX.md`说明文档

```tsx
config ----------------------- // 使用umi配置
  │─config.ts ---------------- // 公共配置文件
  │─defaultSettings.ts ------- // 部分默认配置，暂时无用
  │─proxy.ts ----------------- // 反向代理
src
  ├─app.tsx ------------------ // 运行时文件，主要配置layout以及做路由处理
  ├─services ----------------- // 数据请求
  │ ├─hosts.ts --------------- // 请求域名配置
  │ ├─index.ts --------------- // 接口统一导出，方便引入
  │ └─request.ts ------------- // 请求拦截器，统一错误拦截、token等处理
  │ └─...... -------------------- // 其他模块接口目录
  ├─assets ------------------- // 资源目录
  ├─components ---------------- // 公共组件库 
  │ └─Form -------------------- // 表单相关组件
  │   └─Filter ---------------- // 搜索栏组件
  │   └─UploadOos ------------- // 使用阿里云oos服务的上传组件
  │   └─...... ----------------
  │ └─...... ------------------
  ├─layouts ------------------- // layout部分组件
  ├─locales ------------------- // 国际化文件，主要是针对路由配置，如果不开启国际化，antd组件默认文本是英文状态
  ├─models -------------------- // 约定式全局model
  ├─pages --------------------- // 视图页面
  │ ├─Demo
  │ │ └─UseHooks
  │ │   └─index.tsx
  │ │ └─......
  │ ├─index
  │ │ └─index.tsx
  │ └─......
  ├─router -------------------- // 路由配置
  │ ├─index.ts ---------------- // 各级路由统一导出
  │ ├─System.ts --------------- // 系统管理路由
  │ ├─......
  ├─types --------------------- // 全局性类型声明
  │ ├─index.ts ---------------- // 各个类型声明统一导出
  │ └─......
  ├─utils --------------------- // 工具类库
  │ ├─Hooks ------------------- // Hooks封装
  │   └─useChecked.ts -------- // 多选逻辑封装
  │ ├─default.ts -------------- // 全局可用默认字段表
  │ ├─default_images.ts ------- // 默认图片统一引入/导出
  │ │─pattern.ts -------------- // 正则表达式
  │ ├─index.ts ---------------- // util统一导出
  ├─tests --------------------- // 单元测试
  ├─global.css ---------------- // 全局css
  ├─API.d.ts ------------------ // 全局可用类型声明
  ├─typings.d.ts -------------- // 全局类型ts垫片
```
